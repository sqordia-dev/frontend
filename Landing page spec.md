# Sqordia Landing Page Design Documentation

## Overview

This document provides a comprehensive breakdown of the Sqordia landing page design, structure, and content. The landing page is designed as a high-conversion, single-column layout optimized for guiding visitors from awareness to action in under 60 minutes.

## Design Principles

### Minimalist Aesthetic
- Ample whitespace between sections (py-32 to py-40 on desktop)
- Clean, uncluttered layouts with focused content
- Strategic use of negative space to guide the eye
- Simplified visual hierarchy

### Color Palette

**Primary Color: Deep Strategy Blue (#1A2B47)**
- Used for: Main headings, navigation, footer background
- Represents authority, trust, and professionalism
- Conveys strategic thinking and business acumen

**Secondary Color: Momentum Orange (#FF6B00)**
- Used for: Call-to-Action buttons, highlights, accents
- Represents energy, action, and momentum
- Drives conversion and engagement

**Background Colors:**
- Light AI Grey (#F4F7FA) - Alternating section backgrounds
- White (#FFFFFF) - Primary content backgrounds
- Gradient variations of primary blue for hero sections

### Typography

**Headlines: Plus Jakarta Sans**
- Font weights: 700 (Bold), 800 (Extra Bold)
- Large sizes: 4xl to 7xl (36px to 72px+)
- Used for: All section headings, CTAs, major text

**Body: Inter**
- Font weights: 400 (Regular), 500 (Medium), 600 (Semi-Bold)
- Readable sizes: base to 2xl (16px to 24px)
- Line height: 150% for optimal readability
- Used for: Descriptions, benefits, body copy

### Responsive Design
- Mobile-first approach
- Breakpoints: sm (640px), md (768px), lg (1024px)
- Single-column on mobile, multi-column on desktop
- Touch-friendly button sizes (minimum 44px height)

## Visual Hierarchy & Flow

### Conversion Strategy

1. **Hook (Hero):** Grab attention with bold headline and clear value proposition
2. **Process (How It Works):** Show how easy the journey is (4 simple steps)
3. **Features (Core Modules):** Demonstrate technical capabilities and benefits
4. **Segmentation (Value Props):** Speak to specific audience needs
5. **Trust (FAQ):** Remove objections and build confidence
6. **Push (Final CTA):** Create urgency and provide final conversion opportunity

### Micro-interactions

- **Hover Effects:** Scale up, shadow increase, color changes
- **Animated Gradients:** Subtle pulsing in hero and CTA sections
- **Smooth Transitions:** 300ms duration for all hover states
- **Progressive Disclosure:** FAQ accordions reveal content smoothly
- **Icon Animations:** Icons scale up on card hover

### Accessibility Considerations

- High contrast ratios (4.5:1 minimum)
- Large touch targets (44px minimum)
- Keyboard navigation support
- Semantic HTML structure
- Clear focus states
- Readable font sizes (16px minimum for body text)

## Performance Optimization

### Font Loading
- Google Fonts with display=swap
- Preload critical fonts
- Fallback to system fonts

### Image Strategy
- Use CSS gradients instead of images where possible
- SVG icons for scalability
- Lazy load below-fold content

### Animation Performance
- Use CSS transforms (GPU accelerated)
- Avoid animating expensive properties
- Use will-change sparingly

## Mobile Responsiveness

### Breakpoint Strategy

**Mobile (< 640px):**
- Single column layout

- Larger tap targets
- Simplified navigation
- Full-width cards

**Tablet (640px - 1024px):**
- 2-column grids where appropriate
- Side-by-side CTAs

- Multi-column layouts (up to 4 columns)

## Color Usage Guidelines

### Primary Color (#1A2B47 - Deep Strategy Blue)
- **Use for:** Section backgrounds (hero, CTA, footer), main headings
- **Don't use for:** Small text, secondary elements
- **Pairs well with:** White, light grays, orange accent

### Accent Color (#FF6B00 - Momentum Orange)
- **Use for:** CTAs, highlights, badges, icons, important text
- **Don't use for:** Large background areas, body text
- **Pairs well with:** White, dark blue, light gray backgrounds

### Neutral Colors
- **#F4F7FA (Light AI Grey):** Alternating section backgrounds, card backgrounds
- **#FFFFFF (White):** Primary content backgrounds, card surfaces
- **Gray-300 to Gray-700:** Body text, secondary text, borders

- Hero H1: 5xl to 7xl (48px to 72px)
